## 6. 이관 전 필수 조치

1. 202605의 전체 매출 차이 7,921,196 raw 단위의 원천을 ZC/거래처/상품 코드 단위로 확정한다.
2. 기존 `ZC본부`와 신규 `FC본부`의 귀속 규칙 차이를 확인하고, 챗봇에서 사용할 브랜드 기준을 확정한다.
3. 자재그룹 분석이 운영 기능이면 상품 마스터를 추가 조인하여 `자재그룹`, `자재그룹명`, `기존자재번호`를 채운다.
4. 전환 전 `api/main.py`, `api/action_signals.py`, `api/forecast_engine_v7.py`의 주요 질문 시나리오를 구/신규 뷰로 병렬 실행하여 허용 오차를 합의한다.
5. 과거 이력(2019년)이 필요한 예측/분석 기능은 별도 보존 테이블 또는 기간별 분기 로직을 마련한다.
