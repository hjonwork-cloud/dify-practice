리뷰들에서 말한 '한국적이다', '고즈넉하다'라는 이야기가 대표님 방만 보아도 알 수 있었다.

또 가족사진인 것 같은 액자 하나가 있었는데, 따뜻해 보여서 기분도 좋았다.

#### 2. 이상형 월드컵

많은 반지들을 하나하나 껴보면서 취향을 확인하라고 하셨다.

보기에 마음에 들지 않더라도, 확실한 선택을 위해 모두 껴보라고..

그런 후에, 신랑이 마음에 드는 건 신랑 쪽에 빼두고, 신부가 마음에 드는 건 신부 쪽에,

#### 3. 대표님의 아이들 소개

나머지는 전부 치워버리는 줄 알았더니, 거의 모든 친구들을 설명해주신다.

- 만들게 된 배경

- 영감 받은 것

- 어울리는 피부 톤

- 등등

하나하나 천천히 설명해주시고, 공감해주셨기에 더욱 선택이 수월할 것 같았다.

#### 4. 선택한 이유

내 몸에 닿는 것은 호불호가 강한 사람이기에, 좋은 선택을 하고 싶었다.

우린 제작 의도와, 우리가 원하는 느낌이 아주 잘 맞는 친구로 선택했다.

그리고 생각보다 견적이 괜찮았기에 바로 결정할 수 있었던 것 같다.

또 2년 이내에는 무상 AS로 안심할 수 있고, 작은 다이아가 빠지더라도, 빠져서 잃어버리더라고 OK라는 것..

사실 제품이나 견적이 마음에 들지 않았더라도, 대표님의 따뜻한 마음과 공감 때문이라도, 좋은 기억으로 남았을 것 같다.

조그마한 나의 눈짓, 몸짓을 너무도 잘 캐치하셔서 좋은 선택이 가능했고, 어떤 말이던 좋게 포장해주시던 대표님을 다시 떠올려보면 온전히 본인의 시간과 감정을 진심으로 쏟으시는 게 느껴진다.

사실 당연한 거지만, 안 그런 곳이 있었기 때문에 더 마음이 간 것도 있다.

그래서 우리는 대표님이 평양냉면이라고 소개하셨던 친구로 선택했다.

- 1월 초 가봉(완료)
